# maze_agent

`maze_agent` は Phase 7a の軽量シミュレーションです。

固定された 7x7 迷路で、エージェントがスタートからゴールまで進む経路を BFS で求めます。
ランダム迷路を生成することもできます。
現時点では強化学習ではなく、確実に3D表示できる最小の探索デモです。

## パラメータ

- `grid_size`: 迷路サイズ。ランダム迷路では `7`, `9`, `11` を選べます。
- `steps_per_cell`: 1マス移動を何フレームで表すか。
- `dt`: 1フレームあたりの時間。
- `show_search`: 将来の探索可視化用。現在は結果経路を表示します。
- `randomize`: `true` のときランダム迷路を生成します。
- `seed`: 同じランダム迷路を再現するための番号です。空の場合は毎回変わります。
- `wall_density`: 壁の多さです。大きいほど壁が増えます。

## 観察ポイント

- 壁を避けてゴールへ進むこと
- 経路の長さ
- 最終フレームでゴール座標に到達すること
